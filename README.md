# 灵工保 - 温暖的灵活就业社保服务平台

这是一个基于 [Next.js](https://nextjs.org) 构建的灵活就业社保服务平台，为2亿灵活就业者提供温暖、专业的社保咨询服务。

## 🌟 项目特色

- 🛡️ 温暖设计系统 - 情感化色彩和用户体验
- 🤖 AI智能助手 - 提供社保咨询服务
- 📊 社保状态监控 - 实时跟踪保障状态
- 💰 多种缴费方案 - 灵活选择适合的保障方案
- ⛓️ 区块链存证 - 安全可信的记录保存
- 📱 移动端优化 - 完美的移动端体验

## 🚀 快速开始

首先，运行开发服务器：

```bash
npm run dev
# 或
yarn dev
# 或
pnpm dev
# 或
bun dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看结果。

你可以通过修改 `app/page.tsx` 来开始编辑页面。页面会在你保存文件时自动更新。

## 📁 项目结构

```
├── app/
│   ├── globals.css      # 全局样式和温暖设计系统
│   ├── layout.tsx       # 根布局组件
│   └── page.tsx         # 主页面组件
├── public/              # 静态资源
└── README.md           # 项目说明
```

## 🎨 设计系统

项目采用温暖的设计系统，包含：

- **情感化色彩**: 温暖的橙色主色调，传达关怀和温暖
- **响应式设计**: 完美适配移动端和桌面端
- **微交互动画**: 提升用户体验的细节动画
- **语义化组件**: 清晰的组件结构和命名

## 🔧 技术栈

- **框架**: Next.js 16.0.5 (App Router)
- **语言**: TypeScript
- **样式**: CSS with Custom Properties
- **字体**: Geist (next/font优化)
- **构建工具**: Turbopack

## 📱 功能模块

1. **用户中心**: 展示用户信息和保障状态
2. **社保状态**: 实时监控缴费状态
3. **智能缴费**: 多种缴费方案选择
4. **AI助手**: 智能社保咨询
5. **底部导航**: 便捷的功能切换
